# Zettelkasten Writing Coach
把零散摘录和想法转成可连接、可写作、可生长的卡片盒结构，帮用户从笔记走向文章。

## What it does
- 🧩 原子笔记拆分：将长篇摘录拆成一张卡片一个想法的独立单元
- 🔗 连接关系发现：在卡片之间建立双向链接，揭示隐藏的概念网络
- ✍️ 写作主题提取：从卡片聚类中生成文章题目、核心论点和写作大纲

## Example scenarios

**1. 用卡片笔记法整理读书笔记**
> 👤 我看完了《认知觉醒》这本书，想用卡片笔记法整理一下，怎么开始？
> 🤖 按Zettelkasten方法指导：1）标记3-5个最有感触的观点；2）每观点写一张独立卡片（标签+来源+自己理解+联想）；3）在卡片间建立链接（用Obsidian/Notion/flomo实现）。给出第一张卡片示例，建议每天写2-3张保持可持续。

**2. 整理散乱的摘录**
> 👤 我有一堆关于阅读、记忆、元认知的摘录，不知道怎么整理。
> 🤖 先不按大类归档，而是拆成原子想法：检索比重读更有效、元认知是学习监控能力、连接比分类更重要、输出能倒逼输入。给出连接建议和可形成的文章题目。

**3. 从笔记中发现写作主题**
> 👤 帮我从这些笔记中找出可以写成文章的主题。
> 🤖 主题聚类分析识别可行文章主题，每个附带工作标题、核心论点、支撑笔记大纲和补充研究方向建议。
